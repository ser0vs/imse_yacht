# Welcome to our yacht shop!

## Prerequisites
- docker
- docker-compose
- cloned repository

## How to start

Please enter the main directory ```inse_yacht``` and run the script:

```
sh start_the_app.sh
```

After the building succeeded, please go to the address http://localhost:5001/ in the browser and enjoy using!
